About Me

Hai, saya Dina Nurizka, seorang web developer yang masih belajar untuk merintih karir dalam dunia teknologi. Saya percaya bahwa teknologi memiliki kekuatan untuk mengubah dunia dan saya bersemangat untuk menjadi bagian dari perubahan tersebut. Sejak kecil, saya selalu tertarik dengan dunia teknologi dan komputer. Ketertarikan ini mendorong saya untuk belajar lebih dalam tentang pemrograman dan pengembangan web. Dengan semangat belajar yang tinggi, saya terus mencoba hal-hal baru dan mengasah keterampilan saya di bidang ini. Saat ini, saya sedang fokus untuk meningkatkan kemampuan saya dalam pengembangan web. Saya belajar tentang berbagai teknologi dan bahasa pemrograman seperti HTML, CSS, python JavaScript, PHP dan banyak lagi. Setiap hari, saya mencari peluang untuk belajar dan mengembangkan diri agar dapat menjadi seorang web developer yang kompeten dan berkontribusi secara signifikan dalam industri teknologi.
